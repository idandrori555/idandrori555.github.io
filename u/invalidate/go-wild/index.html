About Fact Check Pro

Fact Check Pro is an experimental application that leverages the Web Speech API for real-time speech-to-text and the Gemini API for fact-checking spoken statements. It's designed to provide quick insights into the veracity of information.

Please note: This tool is for informational purposes and should not replace thorough research or professional advice. The accuracy of fact-checks depends on the underlying AI model and the clarity of the spoken input.
